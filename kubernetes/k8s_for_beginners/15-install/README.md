# Установка Kubernetes

## Требования к ресурсам нод
### Требования к ресурсам control plane node
- CPU — от 2 ядер
- ОЗУ — от 2 ГБ
- Диск — от 50 ГБ

### Требования к ресурсам worker node
- CPU — от 1 ядра
- ОЗУ — от 1 ГБ
- Диск — от 100 ГБ
